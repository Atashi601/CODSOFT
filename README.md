# 💻 Python Projects Collection (5 Tasks)

This repository contains a collection of **5 beginner to intermediate Python projects**.
These projects demonstrate core programming concepts, problem-solving skills, and practical application development.

---

## 📌 Projects Included

### 📋 TASK 1: To-Do List Application

A command-line based application to manage daily tasks.

**Features:**

* ➕ Add tasks
* 📋 View tasks
* ✏️ Update tasks
* 🗑️ Delete tasks
* 🔄 Menu-driven interface

---

### 🧮 TASK 2: Calculator

A simple calculator that performs basic arithmetic operations.

**Features:**

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 🧠 User input-based operation selection

---

### 🔐 TASK 3: Password Generator

Generates strong and random passwords based on user preferences.

**Features:**

* 🔢 Custom password length
* 🔠 Letters, numbers, and symbols
* ⚡ Random secure password generation

---

### 🎮 TASK 4: Rock-Paper-Scissors Game

A fun interactive game between user and computer.

**Features:**

* 🪨 Rock, 📄 Paper, ✂️ Scissors choices
* 🤖 Random computer selection
* 🧠 Game logic implementation
* 🏆 Score tracking
* 🔁 Play again option

---

### 📇 TASK 5: Contact Book

A simple contact management system.

**Features:**

* ➕ Add contact (name, phone, email, address)
* 📋 View contacts
* 🔍 Search contact
* ✏️ Update contact
* 🗑️ Delete contact

---

## 🛠️ Technologies Used

* Python 🐍
* Standard Libraries (random, string)
* Command-Line Interface (CLI)
* Tkinter (for GUI in some projects)
