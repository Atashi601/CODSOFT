# 🧮 Simple Calculator (Python)

This is a basic command-line calculator built using Python. It allows users to perform simple arithmetic operations.

---

## 📌 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* ❌ Handles division by zero
* 🔁 Runs in a loop until user exits

---

## 🛠️ Technologies Used

* Python

---

## ▶️ How to Run

1. Make sure Python is installed on your system
2. Open terminal or command prompt
3. Run the file:

```bash
python calculator.py
```

---

## 💡 How It Works

* User selects an operation (1–5)
* Enters two numbers
* Program calculates and displays the result
* Repeats until user chooses Exit

---

## 📷 Example Output

```
Simple Calculator
1. Add
2. Subtract
3. Multiply
4. Divide
5. Exit

Enter choice: 1
Enter first number: 10
Enter second number: 5
Result: 15
```
